Products

The Products setup is the master list for items your team uses and purchases—seed, chemicals, fertilizer, and grain bags. Keep names, SKUs, suppliers, package sizes, and reference notes consistent so entries across the app stay clean and searchable.

Use the sidebar to open a category and add or edit items. Include labels like rate units, packaging (bag, tote, bulk), and optional details (EPA/registrations, REI/PHI, mixing notes) so operators and office staff stay aligned.

Categories

  • 🌱  Seed — brand/hybrid, trait, RM, seed size, treatment
  • 🧪  Chemical — product name, active, rate units, notes
  • 🧂  Fertilizer — analysis (e.g., 32-0-0), form, density, units
  • 👝  Grain Bags — length/diameter, capacity estimate, vendor

Tips

  • 🏷️  Keep product names short but unique; use notes for details
  • 📦  Store package size + unit (e.g., 2.5 gal, 50 lb, 1000 L)
  • 🔎  Add vendor and SKU to make receipts + reconciliation easier
  • 📝  Use mixing/handling notes so field entries stay consistent
