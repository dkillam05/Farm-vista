Shrink Mode — What it does

Shrink Mode only changes what you see. The calculator always computes both methods.

Modes

  • Compare Both — shows True Shrink and Elevator Shrink side-by-side.
  • True Only — shows only the True Shrink tile.
  • Elevator Only — shows only the Elevator Shrink tile.
